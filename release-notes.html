Maa' v1.0.13

What's New / الجديد في التحديث
الميزات الجديدة والتحديثات 🇸🇦
  • 🎨 أيقونة ملونة لشريط القوائم (Menu Bar)

    خيار جديد في الإعدادات يتيح لك تفعيل أيقونة ملونة أو أحادية اللون لشريط القوائم العلوي لتناسب ذوقك وثيم جهازك.

  • 🌊 أنيميشن تفاعلي لكوب الماء

    تصميم جديد كلياً لكوب الماء داخل نافذة شريط المهام مع مستوى ماء يتحرك بأنيميشن SwiftUI تفاعلي ناعم يعكس كمية استهلاكك اليومية.

  • ⏰ اختيار الوقت الخاص بالنظام (Native Time Picker)

    استبدال طريقة كتابة أوقات التنبيه يدوياً بواجهة اختيار الوقت الأصلية لنظام macOS (Compact DatePicker) لتسهيل ضبط فترات التنبيه بدقة وسرعة.

  • ✨ تأثيرات تفاعلية للأزرار (Hover Effects)

    إضافة تأثيرات حركية (Hover) ناعمة ومحسّنة لجميع الأزرار عند تمرير مؤشر الفأرة عليها لتمنحك شعوراً أكثر تفاعلية واستجابة.

  • 🎉 تهنئة مميزة عند تحقيق هدفك اليومي

    عند اكتمال شرب كمية الماء المستهدفة لليوم، ستظهر لك رسالة تهنئة متحركة مميزة، وستتحول أيقونة شريط المهام لتظهر عليها علامة صح خضراء للإشادة بإنجازك.

  • 🧹 واجهة مستخدم مبسطة وخالية من المشتتات

    حذف زر الغفوة (Snooze) بالكامل لتبسيط الواجهة وتركيز التطبيق على الأفعال الأساسية والهامة فقط.

  • 🌐 دعم كامل ومحسّن للغة العربية

    إصلاح اتجاهات الواجهة والنصوص ودعم الترجمة الكاملة للغة العربية بشكل ديناميكي، مع إعادة هيكلة ملفات الترجمة بشكل منفصل ليسهل على المطورين إضافة لغات أخرى مستقبلاً.

New Features & Improvements 🇬🇧
  • 🎨 Colored Menu Bar Icon Option

    New setting to toggle between a colored and a monochrome menu bar icon to perfectly match your desktop style and preferences.

  • 🌊 Interactive Water Cup Animation

    A completely redesigned water cup inside the menu bar popup, featuring a dynamic fluid level animation using native SwiftUI.

  • ⏰ Native macOS Time Picker

    Replaced manual text entry for settings time intervals with the native macOS compact DatePicker for effortless and precise scheduling.

  • ✨ Premium Hover Effects

    Added smooth hover states and highlight transitions to all action buttons in the menu bar popover for a highly responsive, premium feel.

  • 🎉 Daily Goal Celebration & Badge

    Celebrate hitting your daily water target with a beautiful animated congratulations card. A green checkmark badge will also appear on the menu bar icon when completed.

  • 🧹 Streamlined UI (Removed Snooze)

    Removed the unnecessary Snooze action to simplify user choices and declutter the core menu controls.

  • 🌐 Improved Arabic Localization

    Fully polished layout directions, missing words (e.g. Schedule), and isolated language bundle structures (.lproj) to allow easy language extensions by open-source contributors.
